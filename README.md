## Descripción del proyecto
Este proyecto es un `screensaver` que utiliza las librerías `OpenGL` y `OpenMP`.
## Instrucciones
- Compilar con `g++ main.cpp -fopenmp -lglut -lGL -lGLEW -lGLU -o main.exe`
- Ejecutar con `./main.exe num_objects`
- `./main.exe 5`
